# Movie Data Analysis (Entertainment Analytics)

## Objective
Analyze movie datasets to identify key trends and performance factors in the entertainment industry.

## Solution
- Used MySQL to merge and query multiple related tables containing information on movies, genres, directors, actors, and production houses.
- Extracted insights on:
  - Yearly movie release patterns
  - Top-performing genres based on ratings and earnings
  - Most successful directors and production houses
  - Leading actors for potential casting decisions

## Key Achievement
Identified top genres, high-performing movies, and leading actors that can guide future film project planning.
